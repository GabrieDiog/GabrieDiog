glossário

termos html

<p>
é a tag utilizado para que o navegador saiba interpretar que ali possui um paragrafo
<div>
a <div> é a tag utilizada para alterar o estilo em partes especificas da pagina ou para mudar a posição de objetos
<table>
é a responsavel por sinalizar que ali possui uma tabela
<caption>
tem a função de descrever em forma de texto para o leitor a tabela
